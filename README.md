# Neural Networks From Scratch In Rust
This repository is meant to keep my mind active. I would also like it, if someone used my code as inspiration to create a neural networks library for Rust without any Python dependencies. There aren't many neural networks or deeplearning libraries available for Rust yet, and the one I know uses Python a lot. Using Python adds a needless dependencies and slows down the library. Using pure Rust would still make it easy to use the library, and make it even faster. This could make Rust beter for deep learning than Python currently is, and speedup developments in deep learning even more.
